# DAS 732: Data Visualization - Visual Analytics Workflow

This project focuses on implementing visual analytics workflows for various datasets, including geographical factors, demographic factors, and other related visualizations.

## Directory Structure

- ### A1_Code : The entire A1 assignment codes.
  
- ### Dataset : Contains the dataset files used for analysis.
  
- ### Images  : Contains all the plots plotted during the course of visualization.
  
- ### Demographic_Factors.ipynb : Jupyter Notebook containing the analysis of enrollment data on the basis of demographic factors.
  
- ### Geographic_Factors.ipynb : Jupyter Notebook containing the analysis of enrollment data on the basis of demographic factors.

- ### Temporal_Factors.ipynb : Jupyter Notebook containing the analysis of enrollment data on the basis of demographic factors.

- ### Demographic_Factors.py : Python Source Code containing the analysis of enrollment data on the basis of demographic factors.
  
- ### Geographic_Factors.py : Python Source Code containing the analysis of enrollment data on the basis of demographic factors.

- ### Temporal_Factors.py : Python Source Code containing the analysis of enrollment data on the basis of demographic factors.

- ### report.pdf : Contains the report for this project.

## Setup Instructions

### Prerequisites

#### Ensure Python is installed on your system. You can download it from [python.org](https://www.python.org/downloads/). You'll also need pip to manage Python packages.

### Step 1: Download the Project

### Step 2: Install Dependencies

Install the required Python packages listed in the requirements.txt file:

```pip install -r requirements.txt```

Step 3: Run the Scripts

Navigate to the /scripts folder and execute the desired Python script. For example:

To run Geographical_Factors.py:

```python Grographical_Factors.py```

To run Demographic_Factors.py:
```python Demographic_Factors.py```

To run Temporal_Factors.py:

```python Temporal_Factors.py```

To run the jupyter notebooks, refer (https://jupyter.org/install)
